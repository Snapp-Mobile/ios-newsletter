### [Announcing Swift System Metrics 1.0: Process-Level Monitoring](https://www.swift.org/blog/swift-system-metrics-1.0-released/)

[Vladimir Kukushkin](https://github.com/kukushechkin/){: .author}

### [Sosumi.ai - Making Apple docs AI-readable](https://sosumi.ai)

In their own words:
